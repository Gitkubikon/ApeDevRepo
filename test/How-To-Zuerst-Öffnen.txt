###########	Anleitung	###########

##	1. 	##
Installieren Sie Python (Während der installation bei "Add Python 3.* to PATH" ein Häkchen machen)

##	2.	##
Öffnen sie die Konsole (cmd, powershell)

##	3.	##
Geben sie den Befehl "pip install jpype1 dearpygui" ein und drücken sie die Enter-Taste

##	4.	##
Navigieren sie mit dem Terminal in den Seminar Ordner
Anleitung um im Terminal zu navigieren : https://wiki.communitydata.science/Windows_terminal_navigation

##	5.	##
Geben sie, wenn sie im Seminar Ordner sind, den Befehl "python main.py" ein

###########	Sonstiges	###########

Der Java Code befindet sich in *\Seminar\Java\Main.class
Der Python Code befindet sich in main.py (einfach im Notepad, oder in beliebigem Code Editor öffnen)
Wenn Sie docs.py ausführen, können sie die Dokumentation von "Dear PyGui" lesen
